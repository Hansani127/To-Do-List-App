# 📝 TaskFlow — To-Do List App

A beautiful, fully functional task management web application built with **HTML**, **CSS**, and **JavaScript**. Features user authentication, real-time stats, offline support, and PWA capabilities — no frameworks required!



## ✨ Features

- 🔐 **User Authentication** — Register/Login with secure sessions
- ✅ **Task Management** — Add, edit, complete, and delete tasks
- 📊 **Live Dashboard** — Animated stats cards with progress tracking
- 🔍 **Smart Search** — Real-time task filtering
- 📅 **Due Dates** — Smart labels (Today, Tomorrow, Overdue)
- 🏷️ **Categories & Priorities** — Organize tasks your way
- 🌙 **Dark/Light Theme** — Toggle with persistent preference
- 📱 **PWA Ready** — Install on mobile home screen, works offline
- ⚡ **Smooth Animations** — Slide, fade, pulse, and bounce effects
- ⌨️ **Keyboard Shortcuts** — `Ctrl+K` search, `Ctrl+N` new task
